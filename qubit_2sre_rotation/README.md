# Qubit steady-state 2-SRE versus rotation angle

This folder contains the qubit monitored-circuit simulation for the steady-state 2-SRE response as a function of the rotation angle `theta_M`.

Main files:
- `qubit_2sre_rotation.py`: main simulation script
- `plot_qubit_2sre_rotation.py`: figure replotting with reference slope
- `replot_qubit_2sre_rotation.py`: relabeled plotting script

Data and figure:
- `qubit_2sre_rotation_data.txt`
- `qubit_2sre_rotation.png`
